# Movie App Specification

## Overview
A responsive movie browsing web app that displays a list of movies using card-based UI.

## Features
- Display movie cards with title, poster, rating, and description
- Responsive grid layout
- Reusable card component function
- Simple static data (no backend required)

## Technical Stack
- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript

## Functional Behavior
- Movies are rendered dynamically from a JavaScript array
- Layout adjusts for mobile and desktop screens
- Cards are reusable via a render function
