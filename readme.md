# 🎬 Movie App

A simple responsive movie browsing app built with HTML, CSS, and JavaScript.

## 🚀 Features
- Movie cards with poster, title, rating, and description
- Fully responsive design
- Dynamic rendering using JavaScript
- Clean reusable component structure

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
